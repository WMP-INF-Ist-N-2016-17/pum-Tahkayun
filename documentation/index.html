About Project top

Password Generator is a simple tool used to generate strong passwords. You can use Password Generator to generate strong passwords for your accounts. You have option to select Lower case, Upper case, Symbols and Numbers while generating passwords. Use the button or shake your device to generate new strong passwords

  • Unique passwords
  • Lower case letters
  • Upper case letters
  • Symbols
  • Numbers
  • Shake to generate

Technology Stack

  • Android Studio
  • Java
  • Admob
  • Minimum Android SDK Version: 16

Admob Configuration top

'Password Generator' app is admob ready. If you have tested the app you would noticed the test ads was loading. Please follow the instructions below and ad your own Ad unit ids

Change Ad Unit ID

Goto values/strings.xml at the bottom of this file there is placed admob test unit ids.
ad_appid is the application id from admob. You can get yours from your admob application dashboard. admob_banner, admob_interst are the unit ids for the application.


admob_banner: Banner Ad
ad_interst: Interstitial Ad


You can add or remove ads from the application. Please follow the tutorial on Google Admob page it will be very helpful for you if you have not implemented ads before.
https://developers.google.com/admob/android/quick-start
